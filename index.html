Questions fréquentes

Bon à savoir

Comment obtenir un compte ?
Un administrateur crée votre compte depuis l'application et vous communique votre code d'accès. Pas d'administrateur ? Écrivez à gestionbox@hotmail.com.
Mes données sont-elles sécurisées ?
Oui. Les données sont stockées dans le cloud avec chiffrement, et chaque code d'accès est haché en SHA-256. Personne ne peut le lire en clair.
Ça fonctionne sans connexion ?
Oui. Un cache local permet de consulter vos données hors-ligne ; la synchronisation reprend automatiquement dès le retour du réseau.
Sur quels appareils puis-je l'utiliser ?
Sur iPhone, Android, tablette et ordinateur. Vous pouvez aussi l'ajouter à votre écran d'accueil pour l'utiliser comme une vraie application.
Comment signaler un bug ou proposer une idée ?
Envoyez un email à gestionbox@hotmail.com en décrivant votre besoin et votre appareil. Les bugs sont traités en priorité.
