🔥 Guide Firebase

Activer la synchro
en 5 étapes

Objectif : tout l'équipe voit les mêmes données en temps réel. Tu fais ça une seule fois. Suis les images 👇

1

Ouvre ton projet

Sur console.firebase.google.com, touche ton projet déjà créé :

🔥 Firebase
💧
ELAN GESTION
elan-gestion
👆 touche ici
2

Crée la base « Firestore »

Ouvre le menu de gauche, section Créer (ou Build), puis :

📂 Créer / Build
🗄️ Firestore Database 👆

Puis le gros bouton :

＋ Créer une base de données

➡️ Région : eur3 (europe-west) → Suivant
➡️ Mode production → Activer

3

Colle les « Règles »

En haut de Firestore, touche l'onglet Règles. Efface tout et colle exactement ça, puis Publier :

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /{all=**} { allow read, write: if true; }
  }
}
⚠️ Ces règles sont ouvertes (simple pour démarrer). On les sécurisera après si tu veux.
4

Ajoute une « app Web »

En haut à gauche, touche la roue ⚙️ → Paramètres du projet.

Descends jusqu'à « Vos applications » et touche l'icône </> (Web) :

Vos applications
🍎
iOS
🤖
Android
</>
👆 Web

Surnom : ELAN → Enregistrer l'application. (Ne coche rien d'autre.)

5

Copie la config et envoie-la moi

Une fenêtre affiche un bloc de code. C'est ça qu'il me faut :

Add Firebase SDK
const firebaseConfig = {
  apiKey: "AIza….",
  authDomain: "elan-gestion….",
  projectId: "elan-gestion",
  storageBucket: "….",
  messagingSenderId: "….",
  appId: "1:…."
};
👆 copie tout ce bloc

Touche « Copier » sur ce bloc, reviens dans notre conversation, et colle-le. Je m'occupe du reste.

✅ C'est tout !

Dès que tu me colles le bloc firebaseConfig, je branche la synchro temps réel pour toute l'équipe — chacun verra tout en direct, et l'admin verra tout ce que font les autres.
